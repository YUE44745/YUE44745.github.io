  • 概念
    • IC是把电阻电容二极管晶体管等元器件制作在一个芯片上，具有一定功能的电路。
  • 集成度：每个芯片上的元器件数
  • 摩尔定律：IC集成度每一年半（18个月）翻一翻
  • 特征尺寸（Critical Dimension）：芯片上最小物理尺寸，衡量工艺难度的标志，代表工艺水平。CMOS中mos的沟道长度。bip中接触孔的尺寸。

2.氧化

  • 硅热氧化：氧分子或水分子高温下与硅化学反应，硅表面生长氧化硅。
  • 热氧化目的：SiO2保护硅表面、器件隔离、屏蔽掺杂、形成电介质层。
  • 温度：:750~1100
  • 消耗硅：:45%
  • 因素：
    • 温度：高-->快
    • 时间：x=(bt)^(1/2)
    • 掺杂效应：重掺硅 快于 轻掺硅
    • 晶向：<111>比<100>快
    • 压力：高-->快
    • 方式：湿氧 快于 干氧
  • 掺Cl氧化：氧化系统中通HCl(< 3%),提高质量
    • 优点：Cl离子与界面正电荷中和减少界面电荷积累；氧化前通减少离子玷污
  • 硅的局部氧化（LOCOS,LOCal Oxidation of Silicon）:
    1. 生长垫氧化层
    2. 沉积氮化硅薄膜
    3. 可是局部窗口
    4. 利用SiN阻挡氧化剂分子，来对窗口生长氧化硅
    • 实现器件之间电气隔离
  • SiO2用途：防导电123、保护45、注入67
    1. 栅氧层（热生长）
    2. 层间介质（沉积）绝缘导电金属
    3. 场氧层（热or沉积）限制带点载流子的场区隔离
    4. 保护层（热）划伤和离子玷污
    5. 垫氧层（热）减小应力（SiN/Si）
    6. 注入阻挡层（热）阻挡注入离子
    7. 注入缓冲层（热）注入损伤、沟道效应

3.扩散

  • 扩散：加热，一定种类杂质引入，改变电学性质，N（x）符合要求
  • 目的：相乘pn结
  • B/P/As/Sb
  • 方块电阻：长宽相等的扩散层电阻R＝ R□.L/W，R□ =ρ/Xj
  • 分布：
    • 高斯：表面浓度可控、总量不变、时间越长温度越高-->越深越低
    • 结深：N(x)=NB时交点到表面的距离（横向、纵向）

4.离子注入

  • 较之扩散的优点：可以实现小剂量浅结、深浓度峰等特殊掺杂
  • 概念：高真空复杂系统中，产生电离杂质形成高能离子束，入射到硅片靶中掺杂的过程
  • 优点：
    1. 可以精确控制掺杂浓度和掺杂深度（即N(x)）
    2. 可以获得任意杂质浓度分布（N(x)）
    3. 均匀性、重复性好
    4. 掺杂温度低
    5. 玷污少
    6. 无固溶度极限
  • 注入参数：
    • 注入计量φ：样品表面单位面积注入的离子总数 φ=It/qnA
    • 注入能量：单位（KeV）
    • Nmax=φ/(2pi)^(1/2)detl(Rp)=0.4φ/detl(Rp)
    • 结深：xj=Rp+-detl(Rp)*(2ln(Nmax/NB))^(1/2)
  • 沟道效应：例子未与桂圆子碰撞减速，穿透晶格间隙
  • 解决方法：
    1. 倾斜硅片
    2. 缓冲氧化层
    3. 硅预非晶化：增加Si+注入，低能量（1keV）浅注入
    4. 使用质量较大的原子注入
  • 注入损伤：离子轰击造成晶格损伤（轻离子射的更深，重离子射的浅但角度更大）
  • 注入后退火：1.消除晶格损伤 2.从间隙转换为替位，实现电激活
  • 方法：高温热退火、快速热退火

5.光刻

  • 光刻：将淹没班上的图形精确转移到硅片表面光刻胶膜上的复制过程。
  • 目的：制作电路图形的光刻胶模板，结合后续可是或离子注入，来达到晶圆图形化的目的（光刻胶不是芯片的一部分）
  • 传统正性I线光刻胶溶解于显影液的机理：
    1. 树脂是悬浮于溶剂中的酚醛甲醛聚合物（ 线性酚醛树脂）
    2. 感光剂化合物作为强的溶解抑制剂（不溶解于显影液）被加到线性酚醛树脂中
    3. 在曝光过程中，感光剂（通常为DNQ）发生光化学分解产生羧酸
    4. 羧酸提高光刻胶曝光区域的线性酚醛树脂的溶解度
  • 8个步骤
    1. 气相成底膜：粘附性
    2. 旋转涂胶
    3. 软烘：去除溶剂，粘附性、均匀性、抗蚀性，优化光吸收特性，缓解应力
    4. 对准和曝光：掩膜板上图形通过光学系统传递到光刻胶膜上，形成光敏感物质空间分布，从而实现图形转移
    5. 曝光后烘培（PEB）
    6. 显影：溶解胶膜，形成图形
    7. 坚膜烘培：彻底挥发溶剂，粘附性、抗蚀性
    8. 显影检查
  • 系统组成：
    • 紫外光源
    • 光学系统
    • 掩膜板
    • 对准系统
    • 载片台
  • 数值孔径（NA）:透镜手机衍射光的能力
  • NA=n*sin(θm) n—图像介质的折射率,θm—主光轴与透镜边缘光线的最大夹角,步进和步进扫描式光刻机的NA都能做到0.60～0.68的水平
  • 分辨率R：两个相邻特征尺寸的光刻胶图形分开的能力。
  • R=k*波长/(NA) ,K—工艺因子，范围是0.6～0.8,分辨率是光刻中一个重要的性能指标,提高分辨率（减小特征尺寸）--> 减小波长
  • DOF：焦点上下一个范围，图像连续清晰
  • DOF=波长/2*NA^2 ,1um,NA大波长小 --> 高分辨率、低焦深

6.刻蚀

  • 刻蚀：化学或物理的方法，选择地去除硅片表面层材料的过程
  • 目的：光刻胶图形精确地转移到硅片上，复制掩膜版图形的目的。
  • 各向同性刻蚀：侧向与纵向速度相同
  • 各向异性刻蚀：侧向远小于纵向，侧向几乎不腐蚀
  • 干法较湿法优点：
    1. 各向异性
    2. 均匀性
    3. CD好
    4. 光刻胶不易落
    5. 费用低
  • 干法较湿法缺点：
    1. 选择性差
    2. 损伤
    3. 设备贵
  • 选择比:同一刻蚀条件下，刻蚀一种材料对另一种材料的刻蚀速率之比
  • 高选择比意味:只除想要掉的膜层材料(SiO2)，对其下层材料（Si）和光刻胶不刻蚀。
  • 反应离子刻蚀（Reactive Ion Etch ，RIE ）各向异性刻蚀：
    1. 真空反应室的刻蚀气体在射频电场的作用下分解电离形成等离子体
    2. 系统的电源电极上产生一个较大的自偏置电场
    3. 等离子体中的反应正离子在自偏置电场中加速得到能量轰击样片表面，对样片表面有一定的溅射作用形成物理刻蚀，同时提高了表面层反应元素的化学活性，加速与样片的化学反应
    4. 遭受离子轰击的底面比未遭受离子轰击的侧面的刻蚀快得多，达到了很好的各向异性
  • 氧化硅的刻蚀：接触孔和通孔。
    • CHF3 ＋Ar＋He，RIE系统，≤0.1Torr
    • 物理刻蚀元素Ar＋、CF3＋，化学刻蚀元素CF3
    • H的作用：HF除腐蚀Si的活性基（F原子），提高对下层Si的选择比
  • 多晶硅的刻蚀：形成MOS栅电极，是特征尺寸刻蚀
    • Cl2 ＋Ar，RIE系统，小于0.1Torr
    • 物理刻蚀元素Cl+ 、Ar+ ，化学刻蚀元素Cl
    • 什么不用SF6等F基气体？Cl基气体对下层的栅氧化层（SiO2）有较高的选择比

7.沉积

  • CVD（chemical vapor deposition）：利用电阻加热、等离子体、光辐射等能源使气态物质发生化学反应，生成固态物质沉积在衬底表面形成薄膜

  • 介质层：硅器件与金属层之间或层间介质

  • 层间介质：ILD（InterLayer Dielectric）：金属层与金属层之间的电绝缘层

  • 薄膜特性（IC对薄膜的要求）：

    1. 台阶覆盖
    2. 填充高深宽比间隙
    3. 厚度均匀性
    4. 纯度和密度
    5. 化学剂量
    6. 结构完整和应力低
    7. 电学特性
    8. 粘附性

  • LPCVD（low pressure）SiO2的用途：做ILD、浅槽介质填充和侧墙

  • 工艺：

    • TEOS热分解法（Si（C2H5O4） → SiO2 ＋ H2O ＋ CO2）
    • 650℃～750℃
    • 0.1～5.0Torr
    • 10～15nm/min

  • LPCVD Poly－Si的用途：1）栅电极 2）多晶电阻及桥联 3）PIP电容上下电极

  • LPCVD Poly－Si工艺：

    • 硅烷热分解法（SiH4→ Si＋ 2 H2）
    • 575℃～650℃
    • 0.2～1.0Torr
    • 10～20nm/min

  • 掺杂的Poly－Si做栅电极的原因

    • 特定电阻
    • 优良界面特性
    • 高温工艺的兼容
    • 比金属电极（如Al）更可靠
    • 陡峭结构上沉积均匀
    • 栅的自对准

  • PECVD（Plasma Enhanced）:真空腔，反应气体施加等离子体能量,激发化学活性强的分子、原子，加速反应沉积成膜

  • PECVD SiO2的特点：

    • 温度低，300℃～400℃
    • 速率高
    • 冷壁反应，颗粒少

  • PECVD SiO2的用途:做ILD、浅槽介质填充和侧墙,顶层钝化层

  • LPCVD PECVD
    电阻加热、热壁 等离子体、冷壁
    650~750 250~400
    10~15nm/min 50~60nm/min
    优点 优点+速率高，温度低

    优点：均匀性、颗粒、台阶覆盖

  • 外延的特点

    1. 温度高 1100℃ ～1200℃
    2. 速率高（300nm/min）
    3. 晶格结构、晶向与衬底完全相同
    4. 导电类型、电阻率、厚度可控，与衬底无关

  • 用途：Bipolar、BCD。为了性能更优

8.蒸发与溅射

  • 溅射：在高真空下，利用高能粒子撞击具有高纯度的靶材料表面，撞击出的原子最后沉积在硅片上的物理过程
  • 磁控溅射：利用靶表面附近的正交电磁场,使电子平行靶表面做回旋运动,增加与氩原子的碰撞几率，显著提高等离子体区的Ar离子密度，使溅射速率成倍增加
  • 目的：在芯片上形成金属电极与金属连线用于传输电信号
  • 6个步骤：
    1. 在高真空腔等离子体中产生正氩离子，并向具有负电势的靶材料加速；
    2. 在加速中离子获得动能，并轰击靶；
    3. 离子通过物理过程从靶表面撞击出（溅射）原子；
    4. 被撞击出（溅射）的原子迁移到硅表面；
    5. 被溅射的原子在硅片表面凝聚成膜，薄膜具有与靶相同的材料组分；
    6. 多于粒子由真空泵抽走。
  • 优点：台阶覆盖，金属合金
  • 缺点：速率低
  • 磁控多的优点：速率提高了5～40倍，射频电源能溅射介质，基片温升低（基本上不受电子轰击，二次电子损伤小）
  • 金属化技术术语：
    • 金属化:在绝缘介质膜上沉积金属膜以及随后刻印图形以便形成互连金属线和孔填充塞的过程
    • 互连:用导电材料（如铝、多晶硅或铜）制成的连线用以传输电信号
    • 接触(contact):硅芯片内的器件与第一金属层之间在硅表面的连接
    • 通孔(via):穿过各层介质层从某一金属层到相邻的另一金属层形成电通路的开口
    • 常用的金属种类:铝、铝铜合金、铜、阻挡层金属、硅化物、金属填充塞
  • 阻挡层金属:
    • 提高欧姆接触的可靠性
    • 消除浅结材料扩散或结穿刺
    • 阻挡金属杂质的扩散（如铜扩散）
    • 1.Ti＋TiN , 2.Ta＋TaN（主要用于Cu布线）
  • 硅化物:在高温下难熔金属（通常是钛Ti、钴Co）与硅反应形成的金属化合物（如TiSi2、CoSi2 ）
    • 降低欧姆接触电阻
    • 作为金属与有源层的粘合剂
    • 1.硅化钛TiSi2 , 2.硅化钴CoSi2 （0.25um及以下）
    • 特性:电阻率，电迁徙，工艺兼容
      • 电阻率低（Ti：60 μΩ.cm ，TiSi2 ：13～17μΩ.cm ）
      • 高温稳定性好，抗电迁徙性能好
      • 与硅栅工艺的兼容性好
